# ToamViet Backend - Step 1

## Cài package
npm install

## Tạo file môi trường
Tạo file `.env` trong thư mục `backend/`, rồi copy nội dung từ `.env.example` sang.

## Chạy local
npm run dev

## Test nhanh
- GET `http://localhost:5000/api/health`
- POST `http://localhost:5000/api/auth/register`
- POST `http://localhost:5000/api/auth/login`

## Ghi chú
Step 1 chỉ dựng backend core.
Chưa nối frontend.
Chưa upload ảnh thật.
Chưa có reset password.
