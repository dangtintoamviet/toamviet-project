⚙️ Cần chạy SQL migration một lần
Các cột mới (description, content, seo_title, gallery...) chưa tồn tại trong Supabase của bạn.
Cách fix: Mở Supabase Dashboard → SQL Editor → paste nội dung file supabase-complete-schema.sql → bấm Run.
